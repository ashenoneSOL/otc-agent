## Overview

HashLLM is not just another framework. It is a ground-up reconstruction of autonomous agent architecture, purpose-built for the ElizaOS ecosystem.

No shortcuts. No compromises. No explanations.

## Installation

```bash
git clone https://github.com/ctrlshifthash/.git
cd otc-agent
pnpm install
```

---

## Configuration

```typescript
// hashllm.config.ts
export default {
  runtime: "eliza-native",
  mode: "autonomous",
  verbose: false
}
```

---

## Usage

```bash
pnpm dev
```
